05Projects

TheFutureRomdennes.com

This site was a labor of love created from scratch. I built TheFutureRomdennes.com during the summer of 2013 before proposing to my beautiful wife. I took her for a walk along the same path we shared the night of our first date. When we reached the scenic overlook I gave her an iPad to view a JavaScript slideshow (totally nerdy, yes). When the slideshow ended I popped the question, which she excitedly answered with a yes!

This was the first web project that I built from scratch. I taught myself how to use Bootstrap and built this website to be responsive. I also spent quite a lot of time learning to code JavaScript for that slideshow.


MeisterMadeCrafts.com

My wife likes to make things too – she's a wonderful crafter! In 2016 she created an Etsy shop to share her projects, and we thought it would be a great idea for her to have a professional web presence to go along with it. MeisterMadeCrafts.com is a simple, responsive single-page site built to introduce the shop and showcase a few projects.


LuxemburgCascoSoccer.com

Soccer happens to be my passion, and I’ve worked with Luxemburg-Casco Soccer Club as a coach and fundraising volunteer for several years. In 2012 we decided a new website was in order, and I was the one to build it. In the first iteration I created a brand new web presence from scratch, using Bootstrap, Foundation and PHP. I maintained that structure for two years, until I decided to update the site to Wordpress for easier collaboration and upkeep.


LuxCasco.k12.wi.us

This site is notable for the significant user interface considerations that went into it. The old website was built on a quite dated CMS from the early 2000’s and hadn’t been modernized. Luxemburg-Casco’s District Technology Committee had decided to use the Google Sites platform for its efficient integration with the other tools in the Google Apps for Education suite. The site served L-C’s teachers, students, parents and community members, and with such a wide audience an emphasis on usability was important.

I was tasked with organizing and populating the website on the Google Sites platform. Google Sites isn’t great from a development perspective, and didn't allow for a responsive layout, but I managed to manipulate the platform as needed to create the user interface that I thought fit best for an educational institution. I drew inspiration from other education websites and chose a design that prioritized intuitive navigation.
